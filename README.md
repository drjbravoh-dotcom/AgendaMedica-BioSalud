# Agenda Médica BioSalud

Aplicación para gestión de consultas médicas, pacientes, historia clínica y generación de documentos PDF.

## Funcionalidades:
- Agenda médica para citas
- CRUD de pacientes
- Historia clínica detallada
- Generación de recetas, certificados y órdenes en PDF
- Integración con Firebase

## Instalación:
1. Clonar este repositorio.
2. Configurar Firebase en el archivo `firebase_options.dart`.
3. Ejecutar `flutter pub get` para instalar las dependencias.

## Requerimientos:
- Flutter SDK
- Firebase Project
